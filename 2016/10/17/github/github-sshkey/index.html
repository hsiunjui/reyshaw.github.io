github ssh key的配置

发表于 2016-10-17   |   分类于 github

用于无密码输入的git提交,验证github的需要

设置步骤

登录到github主页

1.右侧图标点击settings
2.选择SSH and GPG keys
3.New SSH key
4.自定义title
5.打开~/.ssh/id_rsa.pub,复制公钥并粘贴到key栏
6.点击add key,输入github密码确认
