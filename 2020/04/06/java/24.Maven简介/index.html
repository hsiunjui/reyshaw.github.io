Maven

Maven是跨平台的项目管理工具,主要服务于基于Java平台的项目构建、依赖管理、项目信息管理。

  • 项目构建
    • 清理
    • 编译
    • 测试
    • 报告
    • 打包
    • 部署
  • 依赖管理
    • jar包之间的依赖
  • 项目信息管理
    • 工程版本控制系统消息
    • 缺陷跟踪系统消息
    • 开发者信息
    • 许可证信息
    • Javadoc
    • 测试覆盖
    • 代码静态分析报告
    • 单元测试覆盖率

Maven相关命令

  • 验证 mvn -validate
  • 编译 mvn -complile
  • 测试 mvn -test
  • 打包 mvn -package(创建jar/war包)
  • 检查 mvn -verify(对集成测试结果检查)
  • 安装 mvn -install(安装打包的项目到本地仓库,以供其他项目使用)
  • 部署 mvn -deploy(到远程仓库)
  • 清除 mvn -clean
  • 自动构建 mvn archetype:generate
