PHP变量的命名规则

  • 变量必须以$打头,后面紧跟变量名称
  • 变量名称必须是以字母或者是_(下划线字符)开始
  • 变量名称只能包含数字字母下划线[A-z0-9_]
  • 变量名不能包含空格
  • 变量名区分大小写
