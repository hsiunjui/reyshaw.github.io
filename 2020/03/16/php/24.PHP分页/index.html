PHP分页操作

发表于 2020-03-16   |   分类于 PHP

PHP分页数据查询及处理.

什么是分页

对较多的记录按一定的数量,一定的条件进行分页展示。

使用场景

从数据库查询出来的的记录,在一页不能完全展示.

分页的形式

  • 直接展示记录的页数
  • 加载更多
  • 瀑布流的自动加载

分页的关键变量

  • current_page,当前所在页
  • total_num,总记录数
  • num,每页展示的记录数

注意点

分页不能超越极限范围.
